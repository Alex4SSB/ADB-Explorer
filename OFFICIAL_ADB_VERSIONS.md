## How to Verify Downloads

### Using SHA-256
```powershell
# Windows (PowerShell)
Get-FileHash -Algorithm SHA256 platform-tools_r35.0.2-windows.zip
```

## Notes

1. **Platform Naming Convention:**
   - `windows` - Microsoft Windows (x86/x64)

2. **Checksum Verification:**
   - Always verify checksums after downloading to ensure file integrity
   - Checksums marked as "To be verified" should be obtained from the official Google repository XML or by downloading from the official source

3. **Latest Version:**
   - Always check the official page for the most recent release: https://developer.android.com/tools/releases/platform-tools

4. **Version Numbering:**
   - Format: `rXX.Y.Z` where XX is major, Y is minor, Z is patch
   - Example: r35.0.2 = revision 35, minor 0, patch 2

## Obtaining Official Checksums
