# Documentation

This directory contains additional documentation for the ADB Explorer project.

## Available Guides

### [Advisory Access Guide](ADVISORY_ACCESS_GUIDE.md)
Comprehensive guide for repository maintainers on how to handle GitHub Security Advisories, especially when facing access issues due to antivirus software or other blocking mechanisms.

**Topics covered:**
- Troubleshooting access to GitHub Security Advisory pages
- Step-by-step process for accepting and publishing advisories
- Alternative CVE assignment processes
- Solutions for the current issue #294
